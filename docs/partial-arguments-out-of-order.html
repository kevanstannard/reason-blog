Partially applying function arguments out of order

Sun Dec 13 2020

ReScript version: bs-platform@8.4.2

Suppose you have the following function:

let divide = (a, b) => a / b

We can partially apply the second argument by specifying an _ for the first argument.

let half = divide(_, 2)

Js.log(half(10))
// 5
