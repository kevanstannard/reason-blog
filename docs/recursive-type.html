How to create a recursive type in ReScript

Mon Dec 14 2020

Use the rec keyword.

Example type:

type rec tree<'a> =
  | Node(tree<'a>, 'a, tree<'a>)
  | Leaf('a)

Example usage:

//      1
//     / \
//    /   \
//   2     5
//  / \   / \
// 3   4 6   7

let numbers = Node(Node(Leaf(3), 2, Leaf(4)), 1, Node(Leaf(6), 5, Leaf(7)))

Example functions:

let isNode = tree =>
  switch tree {
  | Node(_, _, _) => true
  | Leaf(_) => false
  }

let isLeaf = tree =>
  switch tree {
  | Node(_, _, _) => false
  | Leaf(_) => true
  }
